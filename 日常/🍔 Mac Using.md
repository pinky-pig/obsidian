## **1. 文件夹快速在控制台中打开**

`在访达窗口中` -> `按下Option` -> `点击窗口` -> `窗口下面显示文件夹目录` 
-> `在目录上右击` -> `在终端中打开`
 
## **2. 快速新建文件**

> 文件夹在终端中打开

```bash
# 新建文件夹
mkdir test

# 新建文件
touch test.md
```

## **3. VS Code**

快速在 VSCode 中打开文件夹

```bash
code .
```

## **4. 打开隐藏文件夹**


```
 shift + command + .
```
